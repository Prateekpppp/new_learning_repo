To run the Project following are the steps 

- For frontend
    -- open the userFeedback folder in vscode or in terminal
    -- run (npm install) to install dependencies
    -- run (npm run dev) in terminal
    -- you will get the localhost url to run the app

- For backend
    -- open the userFeedback folder in vscode or in terminal
    -- run (npm install) to install dependencies
    -- run (npm run start) in terminal to run the server
    -- need to replace the db connection string at line 14 in server.js file
